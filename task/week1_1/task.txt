Цель - создать агента-собеседника, который умеет отвечать на вопросы 
пользователя о современной истории России.

Например работа программы может выглядеть так(на примере консольной программы) -
агент > Пожалуйста введите ваш вопрос
пользователь > В каком году был избран президент Ельцин?
агент > Ельцин был избран в таком-то году и т.д 
агент > Пожалуйста введите ваш вопрос
и т.д

Требования к реализации -
1. Агент должен быть разработан на языке программирования golang в виде консольной
программы

2. Агент делает запросы в облачный ИИ - API GigaChat.

3. После получения ответа от API агент выводит этот ответ в виде 
текста в терминал. 

4. Запуск агента происходит стандартно как для любой другой консольной утилиты,
например -
> ./modernRussiaHistoryAgent

5. В процессе реализации стоит использовать максимально простые решения, 
запрещается использовать любые сторонние библиотеки, все необходимо делать 
средствами стандартной библиотеки golang.

6. Тесты для кода не писать без моего приказа.



Как использовать API GigaChat? 
1. Имеем предварительно полученный токен для авторизации, 
OAUTH_TOKEN. OAUTH_TOKEN действует бессрочно.

2. Используя OAUTH_TOKEN, делаем запрос для получения REQUEST_TOKEN, 
REQUEST_TOKEN действует 30 минут. Пример рабочего запроса посмотри в файле
oauth_post.sh.

3. Используя REQUEST_TOKEN делаем запрос, пример запроса посмотри в 
completions_post.sh



Дополнительные указания
1. RqUID поле оставь как есть
2. Сейчас не нужно обновлять REQUEST_TOKEN когда время его действия истечет,
пусть программа пока-что будет работать 30 минут.
3. В файле network.go должно лежать все что связано с выполнением запроса в сеть.
    В файле agent.go должна лежать логика работы агента(взял пользовательский ввод и 
сделал запрос в сеть на их основе)
    В файле main.go пусть лежит логика получения пользовательского ввода-вывода.
4. Не делай дополнительных проверок пользовательского ввода - считай что 
пользователь всегда вводит корректные данные.
5. Код должен быть максимально простым, понятным, немногословным.
